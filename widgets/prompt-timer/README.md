# prompt-timer

The write-timer engine plus a random **writing-prompt image**: tap to reveal a fresh image from `assets/prompts/` (fetched at runtime via the GitHub contents API). A writing-inspiration tool, distinct from the plain timer.

Self-contained — open `index.html`, or embed via `https://klossgott.github.io/infocusapp/widgets/prompt-timer/` (16:9 iframe).
