# voiceblankpage

**Voice of the Blank Page (VOBP)** course material. `courage-prompts.html`, `curiosity-prompts.html`, `presence-prompts.html` are themed writing-prompt decks (one random prompt per tap); `curiosity.html` is a side-by-side container embedding a timer + the curiosity deck for timed writing.

Self-contained — open `index.html`, or embed via `https://klossgott.github.io/infocusapp/widgets/voiceblankpage/` (16:9 iframe).
