# timer

Canonical write timer. A circular SVG ring counts down a preset duration over a looping piano track (`../../assets/audio/`), with click-to-pause and a 5-second fade-out at the end. This is the base engine the other timers derive from.

Self-contained — open `index.html`, or embed via `https://klossgott.github.io/infocusapp/widgets/timer/` (16:9 iframe).
