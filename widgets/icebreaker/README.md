# icebreaker

Smash-the-ice-cube game: a mallet breaks an ice cube (with sound) to reveal a prompt underneath. Assets (cube/mallet/broken-ice PNGs + chime) live in this folder.

Self-contained — open `index.html`, or embed via `https://klossgott.github.io/infocusapp/widgets/icebreaker/` (16:9 iframe).
