# countdown-timer

A simpler, distinct timer (not the SVG-ring engine): type any number of minutes, get a Web-Audio synth chime and a small celebration animation at zero. `index.html` is the dark theme; `white.html` is the light-on-dark variant.

Self-contained — open `index.html`, or embed via `https://klossgott.github.io/infocusapp/widgets/countdown-timer/` (16:9 iframe).
